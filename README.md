# Employee-Tracker
Employee Tracker 
 # Usage 
 Command-line application that allows the user to:

 *  Add departments, roles, employees
 *  View departments, roles, employees
 *  Remove Employee

 # Technologies 
*  Mysql 
* InquirerJs
